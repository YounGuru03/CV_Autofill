# Skill Matcher

Identify skill gaps between your current skills and job requirements. Get personalized learning recommendations and track your skill development.

## Features

- **Skill Gap Analysis**: Compare your skills against job requirements
- **Match Score**: Visual percentage match with color-coded feedback
- **Learning Path**: Prioritized list of skills to learn
- **Resource Recommendations**: Curated learning resources
- **Export & Save**: Download analysis or save to browser
- **Role Templates**: Quick-load common job requirements

## Usage

1. Open index.html in your browser
2. Enter your current skills
3. Paste job requirements or select a role template
4. Click "Analyze Skill Gap"
5. Review match score and learning recommendations

## Browser Compatibility

Chrome/Edge 90+, Firefox 88+, Safari 14+, Opera 76+
