# Job Search Tracker

Track all your job applications in one organized dashboard. Monitor status, response rates, and never miss a follow-up.

## Features

- **Application Tracking**: Log company, position, status, dates
- **Statistics Dashboard**: Total apps, interviews, offers, response rate
- **Status Management**: Applied, Screening, Interview, Offer, Rejected
- **Search & Filter**: Find applications quickly
- **Notes & URLs**: Save job links and personal notes
- **Export**: Download as CSV for backup or analysis
- **localStorage**: All data saved in your browser

## Usage

1. Open tracker.html in your browser
2. Click "+ Add Application" to log a new job
3. Fill in company, position, and status
4. Track your progress in real-time
5. Export data anytime

## Status Options

- **Applied**: Submitted application
- **Screening**: Phone/recruiter screen scheduled
- **Interview**: Technical/onsite interviews
- **Offer**: Received job offer
- **Rejected**: Not moving forward

## Data Storage

All data is stored locally in your browser using localStorage. No server needed.

## Export

Click "Export" to download all applications as CSV for spreadsheet analysis or backup.
